Q1. What are you really trying to do? What are the goals for my project?

* Build a whatsapp automation to be used personally for my Shopify store  
* I want to send whatsapp template messages to customers of different events like Prepaid Order Confirmed, COD Order Confirmation, Order Fulfillment, Abandoned Cart etc.  
* I also want to have a conversation tab similar to whatsapp where I can see all the conversation and the messages of it which includes text, images, videos and audios  
* I want to see the list of all customers and also have the ability to create a customer with inputs like city, email, tags etc.

Q2. What are the milestones of functionality you want?

| Version | Core App Functionality |
| :---- | :---- |
| MVP | Build a dashboard to handle the different functionalities with basic analytics. Create a page  where I can create whatsapp templates and send it for approval to Meta. Segregate them based on Approval, Pending, Failed. Also follow proper Meta standards Create a page for creating automations and linking it with a whatsapp template. The events will be of Shopify and based on that the templates will be triggered. Create a conversation/chat page where we can see all the conversations with the customers and all the messages including media types. Again will be following proper Meta standards. Create a page for viewing the list of all customers in a tabular format. Adding new customer with specific details |
